About Icarus

ICARUS is an abbreviation for ‘Integrating Computerized ARchitecture with USers’. The aim of the Icarus Project is to develop an open source platform for computer programming implemented in architecture, for teaching and research.

In particular, the platform provides the framework for a simplified and user friendly textual programming methodology for the needs of architectural institutions all over the world. It consists of several publicly available modules like coding, plug-in and repository development.

Grasshopper

A popular algorythmic modeling plugin for Rhinoceros 3D with Python support, Grasshopper has been integrated into the latest version of Rhino. You can find Icarus' codebase for Grasshopper in the Github repository.

Other Platforms

The Icarus project aims to expand to cover the widest possible range of design platforms in the architecture community. Keep checking back back here for updates on new platforms that the Icarus project supports or you can join the Icarus community on Github to help expand Icarus' reach!

Contribute

Icarus was envisioned by its creators, Dr. Odysseas Kontovourkis and Constantine Doumanidis, as a free and open platform built by the community for the community. All of Icarus' source code, tools, documentation and other materials are available completely free under open source licenses.

Want to improve your skills? Inspire and get inspired? Meet other people people with similar interests from all over the globe? Find a mentor or teach someone? Give back to the architecture community? Build something that might help you advance your career? Come join the Icarus community!

There are so many ways to contribute to Icarus! For example you can help translate an Icarus project to your native language benefiting your local architecture community. Into graphic Design? We could always use a hand in improving graphics and user interfaces in the community's projects. Are you honing your programming skills for architecture projects? You can start or expand an Icarus coding subproject. Wherever your talents or interests lie, the Icarus community is ready to welcome you!
